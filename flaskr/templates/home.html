Online Course Manager — OCM

E-Course management platform. Organise online courses efficiently with automatic assignment checking, student/teacher user control and self-enrolment.

Get Started!

OCM Features

Current and upcoming features:

Student:
  • View and Enroll on Courses
  • Read course material straight from the course page
  • Complete assigned exercises
  • Track your progress on summary page
Teacher:
  • Create courses and exercises
  • Keep track of students progress
  • Post announcements visible to students
  • Manage/Control participants and course's invitation code

Open Source

This project is made for University of Helsinki's Database Application-course and is MIT licensed. More info and local building instructions can be found from the GitHub repository.
